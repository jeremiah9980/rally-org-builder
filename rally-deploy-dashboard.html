API — registry 0 next —
Every time rally-ship.sh --dashboard-repo <owner/repo> deploys a site, it appends that repo to registry.json here. The dashboard merges the registry with your manual list below, so new sites appear on their own. A read-only token is needed to read a private registry repo.
Without a token you get ~60 API calls/hour (public repos only). A fine-grained PAT scoped read-only to Actions, Pages, and Metadata raises it to 5,000/hr and shows private repos. The token is saved in this browser's localStorage, sent only to api.github.com, and never written into the file.
